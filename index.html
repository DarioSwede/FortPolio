Byt lösenord

Ange ett nytt lösenord. Din data krypteras om direkt i webbläsaren - inget skickas någonstans utom till GitHub (om du anger en token nedan), som sparar den nya koden i js/data.js åt dig automatiskt.

Skapa en fine-grained token med enbart Contents: Read and write för darioswede/fortpolio på github.com/settings/personal-access-tokens/new. Utan token visas koden istället för manuell kopiering.
